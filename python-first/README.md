Word Count - The first (Python) example for spark
========================================

1. Add pyspark to your PYTHONPATH
```
export PYTHONPATH=$PYTHONPATH:~/osrc/incubator-spark/python # a location in your file system
```

2. Run the example
```
python SimpleApp.py
```
